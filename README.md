# MATLAB-IMAGE-EDITOR
Basic image editor using MATLAB 

##Featuers 
1- Brightness.<br>
2- Contrast.<br>
3- Resolution.<br>
4- Resize.<br>
5- Rotation.<br>
6- Crop.<br>
7- Color Maps (applies only to grayscale images).<br>
8- YIQ and HSV (applies only to truecolor images).<br>
9- Unlimited Undo and Redo.<br>
10- Transformation functions:
  - rgb2gray
  - gray2rgb
  - rgb2ind
  - ind2rgb
  - gray2ind
  - ind2gray
<br>
